Thoghts on Progressive Enhancement

Progressive enhancement is the concept of working from the most basic layer and building up to the more complex version later. Instead of the old method of building for the most complex and then taking care of the older browser versions later. You build from the content up and add css and javascript to make it more interesting. This is a better approach because it does not leave out a large ammount of your user base. It is important because it allows you to build the most accessible websites that can be enjoyed by the most people. Progressive Enhancement
